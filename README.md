# DriftaBot CLI

## Installation

```bash
brew tap DriftaBot/cli
brew install driftabot
```

## Usage

After installation, you can run driftabot from the command line:

```bash
driftabot --help
```

For more information, visit the [main repository](https://github.com/DriftaBot/agent).
